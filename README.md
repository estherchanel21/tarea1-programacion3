# README – Proyecto: Identificar tu Generación

Este proyecto es un programa muy sencillo en Python que pide al usuario su **nombre** y su **edad**, y luego le dice a qué **generación** pertenece.

El programa funciona así:

1. Te pide que escribas tu nombre.
2. Te pide que escribas tu edad.
3. Según la edad el programa dice si eres:

   * Generación X
   * Milenial
   * Generación Z
   * Generación Alfa
